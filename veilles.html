Introduction

La veille technologique consiste à surveiller de manière fréquente les évolutions et innovations d’un domaine / sujet. Elle permet à un développeur de rester informé des avancées, d’anticiper les changements et de maintenir ses compétences à jour dans un secteur en constante évolution comme l’informatique.

Mon sujet : Blazor — Le framework web de Microsoft

J’ai choisi de faire ma veille technologique sur Blazor, le framework web open-source de Microsoft permettant de créer des applications web interactives en C# et .NET, sans avoir recours à JavaScript. Blazor repose sur WebAssembly (Blazor WebAssembly) ou sur le rendu côté serveur (Blazor Server), offrant ainsi une grande flexibilité selon les besoins du projet.

Cette technologie représente une évolution majeure pour les développeurs .NET : elle permet de partager du code entre le front-end et le back-end, de réutiliser des composants, et de s’appuyer sur l’écosystème riche de .NET. Des entreprises utilisant déjà la stack Microsoft trouvent dans Blazor une solution naturelle pour moderniser leurs applications web.

Ce sujet me permet de suivre l’évolution du développement web dans l’écosystème Microsoft, de comprendre les enjeux de WebAssembly et d’approfondir mes compétences en C# et .NET.
